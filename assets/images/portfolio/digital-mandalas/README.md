# Digital Mandalas Images

## Folder Structure

- **`/display/`** - Compressed images for website gallery (JPEG, 800x800px, ~150KB)
- **`/originals/`** - High-resolution originals for downloads (PNG, 4000x4000px+)

## How to Add New Mandala

1. **Save your high-res PNG to `/originals/`**
   - Filename: `mandala-name.png`
   - Size: 4000x4000px or higher
   - Format: PNG (lossless)

2. **Create compressed version for `/display/`**
   - Filename: `mandala-name.jpg` (same name, different extension)
   - Size: 800x800px
   - Format: JPEG
   - Quality: 75-80%
   - Max size: ~150KB

3. **Update `portfolio.html`**
   - Add new portfolio item with image paths

## File Naming

Use descriptive, lowercase names with hyphens:
- ✅ `mandala-lunar-symmetry.jpg`
- ✅ `mandala-cosmic-bloom.jpg`
- ❌ `Mandala 1.jpg`
- ❌ `NEW_MANDALA.jpg`
