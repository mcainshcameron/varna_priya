# Blog Posts

## How to Add a New Blog Post

1. **Copy the template**
   - Use `post-template.html` as starting point

2. **Rename the file**
   - Format: `YYYY-MM-DD-post-title.html`
   - Example: `2024-03-15-creating-cosmic-mandalas.html`

3. **Add featured image**
   - Save to: `/assets/images/blog/featured/`
   - Filename: `blog-YYYY-MM-DD-post-title.jpg`
   - Size: 1200x675px (16:9)

4. **Add content images** (if needed)
   - Save to: `/assets/images/blog/content/`
   - Filename: `post-title-image-1.jpg`

5. **Edit the post content**
   - Update title, date, text
   - Add your images with proper paths

6. **Update blog.html**
   - Add new blog card to the grid
   - Link to your new post
   - Use featured image for card

## File Naming Convention

```
2024-03-15-creating-mandalas.html
2024-04-01-finding-cosmic-inspiration.html
2024-05-10-glass-painting-techniques.html
```
