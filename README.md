# Varna Priya

**Where the cosmos meets the craft table**

A static website for cosmic art, mandalas, and handcrafted celestial pieces.

## Live Site

https://mcainshcameron.github.io/varna_priya/

## Features

- Peacock feather color palette
- Portfolio gallery with filtering
- Blog system
- Free resources downloads
- Ko-fi integration
- Fully responsive

## Folder Structure

```
assets/images/
  ├── hero/                    # Homepage hero image
  ├── portfolio/               # Portfolio images
  │   ├── digital-mandalas/    # display/ + originals/
  │   ├── physical-crafts/     # display/ + originals/
  │   └── celestial-wallpapers/# display/ + originals/
  └── blog/                    # Blog images

blog/posts/                    # Blog post HTML files

downloads/
  ├── free-resources/          # Free downloads
  └── preview/                 # Preview thumbnails
```

## Bot Management

See `BOT-COMMANDS.md` for Telegram bot commands to manage content.

## Deploy to GitHub Pages

1. Push to repository
2. Go to Settings → Pages
3. Select main branch
4. Save

## Customize

- Update social links (Instagram, Ko-fi) in HTML files
- Add your images to `assets/images/`
- Add downloads to `downloads/free-resources/`
- Edit colors in `css/style.css`
