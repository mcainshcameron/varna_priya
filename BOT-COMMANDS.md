# Quick Bot Commands for Varna Priya

Simple reference for managing the website via Telegram/OpenClaw.

## 📝 Blog & Content

| What you want | Say this | Bot does |
|---------------|----------|----------|
| New blog post | "Create blog post about [topic]" | Writes post, creates HTML, updates blog page |
| Add image to post | Upload image after creating post | Adds featured image to blog post |
| New post with image | "Blog post about [topic]" → upload image | Creates post with your image |

**Example:**
```
You: "Create blog post about my creative process"
Bot: Creates 2024-03-15-creative-process.html
You: [upload image]
Bot: Adds image and publishes ✅
```
