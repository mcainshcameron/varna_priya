# Free Resources - Downloads

This folder contains all free downloadable content for your community.

## Folder Structure

- **`/wallpapers/`** - Desktop and mobile wallpapers
- **`/mandalas/`** - Printable mandalas (PDFs for coloring/meditation)
- **`/patterns/`** - Seamless patterns for design projects
- **`/templates/`** - Design templates (Instagram stories, social media)

## How to Add a New Free Resource

### 1. Create Your Resource
- Wallpaper: 3840x2160px (desktop) or 1080x2400px (mobile)
- Mandala PDF: 300 DPI, print-ready
- Pattern: Seamless/tileable PNG
- Template: PSD or PNG format

### 2. Save to Appropriate Folder
```
downloads/free-resources/wallpapers/wallpaper-starlight-dreams.png
downloads/free-resources/mandalas/mandala-coloring-pack-01.pdf
downloads/free-resources/patterns/pattern-cosmic-stars.png
downloads/free-resources/templates/instagram-story-template.psd
```

### 3. Create Preview Image
- Size: 400x300px
- Format: JPEG
- Save to: `downloads/preview/preview-[filename].jpg`

Example:
```
Resource: downloads/free-resources/wallpapers/wallpaper-starlight-dreams.png
Preview: downloads/preview/preview-wallpaper-starlight-dreams.jpg
```

### 4. Update resources.html
Add new resource card with:
- Preview image path
- Download button linking to actual file
- Title and description

## File Naming

Use descriptive, lowercase names:
- ✅ `wallpaper-nebula-nights.png`
- ✅ `mandala-mini-pack.pdf`
- ✅ `pattern-celestial-set.png`
- ❌ `Wallpaper 1.png`
- ❌ `NEW FILE.pdf`

## File Size Guidelines

- **Wallpapers:** 1-5MB (PNG)
- **PDFs:** 1-10MB (compressed)
- **Patterns:** 500KB-2MB (PNG)
- **Templates:** Variable (compress when possible)
