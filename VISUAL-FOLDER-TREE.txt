TYPICAL FILE LOCATIONS
======================

NEW HERO IMAGE:
→ assets/images/hero/new-image.jpg
  Then update: index.html

NEW PORTFOLIO MANDALA:
→ assets/images/portfolio/digital-mandalas/display/mandala-name.jpg (compressed)
→ assets/images/portfolio/digital-mandalas/originals/mandala-name.png (high-res)
  Then update: portfolio.html

NEW BLOG POST:
→ blog/posts/2024-XX-XX-post-title.html (copy from template)
→ assets/images/blog/featured/blog-2024-XX-XX-post-title.jpg
  Then update: blog.html

NEW FREE WALLPAPER:
→ downloads/free-resources/wallpapers/wallpaper-name.png
→ downloads/preview/preview-wallpaper-name.jpg
  Then update: resources.html


WORKFLOW SUMMARY
================

1. CREATE CONTENT → Save to appropriate folder
2. ADD PREVIEW → If downloadable resource
3. UPDATE HTML → Link to new content
4. GIT PUSH → Upload to GitHub
